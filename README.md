## Project features
Three levels of users:
1. Customer
    * search for products
    * purchase products
    * add/remove items from cart
    * checkout

2. Vendor
    * Add/delete items to sell
    * Upload image of items
    * Adjust item information (price, quanitity, description, etc.)

3. Admin
    * Delete customer
